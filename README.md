# Сучасний музичний світ України

Веб-застосунок, який візуалізує та аналізує дані про українських музичних виконавців, використовуючи відкриті дані з **Wikidata**.

Проєкт автоматично завантажує інформацію про артистів, їхні нагороди, дати та місця народження за допомогою SPARQL-запитів, фільтрує некоректні дані та надає зручний інтерфейс для пошуку та сортування.

## ✨ Особливості

* **Динамічне завантаження даних:** Отримання актуальної інформації з Wikidata в реальному часі за допомогою SPARQL.
* **Аналітика та Статистика:**
    * Підрахунок загальної кількості знайдених виконавців.
    * Підрахунок сумарної кількості нагород.
    * Розрахунок середнього показника нагород на одного виконавця.
* **Пошук та Сортування:**
    * "Живий" пошук за іменем виконавця (миттєва фільтрація).
    * Сортування за кількістю нагород, за алфавітом (з коректною обробкою регістру А-Я) та за віком.
* **Якість даних:**
    * Автоматична фільтрація технічних ідентифікаторів (Q-codes), якщо у виконавця немає зрозумілого імені.
    * Підтримка англійської мови як резервної, якщо немає перекладу українською.

## Технологічний стек

* **Backend:** Python 3, Flask
* **Data Source:** Wikidata Query Service (SPARQL)
* **Libraries:** `SPARQLWrapper` (для запитів), `Werkzeug`
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS, Fetch API)
